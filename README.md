# 夏休み勝利ログ

夏休み中の生活ルールを毎日記録し、勝利日数・連続勝利日数・勝利率・平均点を見える化するシンプルなWebアプリです。

## 使い方

1. `index.html` をスマホまたはPCのブラウザで開きます。
2. 今日の項目をチェックします。
3. チェック内容は即時に `localStorage` へ保存されます。
4. ダッシュボードと履歴に達成状況が反映されます。

スマホで毎日使う場合は、GitHub Pages、Netlify、Cloudflare Pagesなどの静的ホスティングにこのフォルダを置き、スマホのブラウザでURLを開いてホーム画面に追加するのがおすすめです。

## 勝利日の条件

以下の2つが両方チェックされている日を勝利日として扱います。

- 8:25 出発
- 22:00 就寝

`6:00 起床`、`SNSは移動中のみ`、`夜作業なし` は加点項目です。1項目1点で、1日5点満点です。

## ご褒美ルール

夏休み終了時点で勝利率90%以上なら、贅沢1万円券を解禁します。

使い道はタクシー、レストラン、グリーン券、欲しいモノ、カフェなど自由です。
参考書、事業用ガジェット、投資資金、AI課金、予備校・勉強の必要経費には使いません。

## 設定

夏休み期間は `app.js` 上部で変更できます。

```js
const START_DATE = "2026-07-09";
const END_DATE = "2026-08-31";
```

## データ

保存キーは `summerRoutineLog` です。JSONエクスポート、JSONインポート、全記録リセットに対応しています。

データは開いている端末・ブラウザ内に保存されます。同じURLを同じブラウザで開く限り残りますが、別端末との自動同期はありません。機種変更や別ブラウザへ移す場合はJSONエクスポートとJSONインポートを使ってください。
